# Assets Directory

Esta carpeta contiene los recursos visuales de la aplicación Celera.

## Logo

Para añadir el logo de Celera:

1. Coloca el archivo de imagen PNG en esta carpeta
2. Renómbralo a `logo.png`
3. Reinicia la aplicación

### Especificaciones recomendadas del logo:
- Formato: PNG con fondo transparente
- Tamaño: 200x200px o superior (se ajustará automáticamente a 60px de altura)
- Proporción: Cuadrada o rectangular horizontal

Si no se proporciona un logo, se mostrará un emoji por defecto.
